# kindle-root-password - Decrypt your Kindle root password

# ABOUT

`kindle_root_password.py` calculates your Kindle root password based on your serial number.

# REQUIREMENTS

* [Python](http://python.org)

# EXAMPLE

    $ python kindle_root_password.py 1234
    fiona2ca0
